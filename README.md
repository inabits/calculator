# 🧮 Calculator App

Uma aplicação web de calculadora simples, intuitiva e responsiva desenvolvida para realizar operações matemáticas básicas do dia a dia. Este projeto foi idealizado para praticar conceitos de manipulação do DOM (Document Object Model), lógica de programação e estilização moderna.

## ✨ Funcionalidades

- **Operações Básicas:** Adição, Subtração, Multiplicação e Divisão.
- **Funções Especiais:** Limpar exibição (AC/C), inversão de sinal (+/-) e cálculo de porcentagem (%).
- **Interface Responsiva:** Design adaptável para telas de smartphones, tablets e computadores.
- **Suporte a Teclado (Opcional):** Permite inserir números e operadores diretamente pelo teclado físico.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web padrão, sem dependências externas:

* **HTML5:** Estruturação semântica dos botões e do visor.
* **CSS3:** Estilização moderna (utilizando CSS Grid/Flexbox) e transições suaves.
* **JavaScript:** Lógica matemática e gerenciamento de estados dos eventos de clique.

## 📦 Como Executar o Projeto

Para clonar e executar este projeto localmente no seu computador, siga os passos abaixo:

1. Clone o repositório:
   ```bash
   git clone https://github.com/inabits/calculator.git
   ```
   
2. Entre na pasta do projeto:
   ```bash
   cd calculator
   ```

3. Abra o arquivo `calculator.html` em qualquer navegador moderno.

Ou simplesmente dê um duplo clique no arquivo `calculator.html`.
